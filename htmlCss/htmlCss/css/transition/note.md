### Transition

1. transition:
2. transtion-property: <name property>
3. transtion-duration: 0.2s;
4. transtion-timing-function:
   1. ease(default): chậm nhanh chậm
   2. linear: bằng nhau
   3. ease-in: nhanh dần
   4. ease-out: chậm dần
   5. ease-in-out: slow start and end
   6. cubic-bezier(n, n, n, n)
